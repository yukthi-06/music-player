# User Prompt
```
Existsing Settings page which contains folder information - its menu is to be renamed to "Folders..."
```

# Implementation Plan - Rename Settings Menu to Folders Configuration

This document outlines the detailed steps to rename the drawer navigation menu item and the corresponding activity title from "Settings" to "Folders...".

## 1. Update String Resources (strings.xml)
- Change `<string name="menu_settings">Settings</string>` to `<string name="menu_settings">Folders...</string>`.
- Since the drawer navigation layout `menu_drawer.xml` references `@string/menu_settings` dynamically, this instantly updates the text displayed inside the drawer menu.

## 2. Update Action Bar Title (SettingsActivity.java)
- Change `getSupportActionBar().setTitle("Settings")` to `getSupportActionBar().setTitle("Folders...")`.
- This ensures consistency between the user-facing navigation item and the header of the page they arrive on.
