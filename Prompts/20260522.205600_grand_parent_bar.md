# Implementation Plan - Hierarchical Grand Parent Folder Navigation

This document outlines the detailed steps to add a "Grand Parent Folder" bar container to browse musical files in a structured, hierarchical parent-child directory model.

## 1. UI Integration (activity_main.xml)
- Insert a new layout bar (`bar_grand_parent_container`) between the top **Playlist Bar** and the **Folder Bar**.
- Incorporate previous/next navigation buttons (`btn_grand_parent_left`, `btn_grand_parent_right`) to switch between grand parent directories.
- Display a small `"GRAND PARENT FOLDER"` section badge inside a vertical layout, with the active grand parent's folder name (`txt_grand_parent_name`) positioned beneath it.

## 2. Parsing & State Initialization (MainActivity.java)
- Define `barGrandParent` and `txtGrandParent` views and reference them in `initViews()`.
- Add `grandParents` path list and `currentGrandParentIndex` state variables.
- Write `buildGrandParentList()` to parse the loaded folders during scan completion and compile unique absolute path strings of their parent directories (grand parent folders).

## 3. Gestures & Navigation Algorithm (MainActivity.java)
- Bind swipe gestures (left/right to switch grand parents) and click events on the Grand Parent bar inside `setupGestureDetectors()`.
- Implement `nextGrandParent()` and `prevGrandParent()` bounds checking.
- Implement `selectFolderFromGrandParent()` to automatically identify the first child folder belonging to the selected grand parent folder, setting `currentFolderIndex` to that index and playing its contents to provide seamless nested traversal.
