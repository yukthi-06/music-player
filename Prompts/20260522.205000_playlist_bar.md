```
Make the top bar for playlist.
Also show in small font what each bar is for.. viz.. Playlist, Folder, Files
```

# Implementation Plan - Reorder Playback Bars & Add Type Labels

This document outlines the detailed steps to rearrange the playback bar containers and introduce visual type badges to easily identify what each bar manages.

## 1. UI Layout Rearrangement (activity_main.xml)
- Reorder the three horizontal navigation containers (`LinearLayout` widgets) such that:
  - **Playlist Bar** (`bar_playlist_container`) is at the top.
  - **Folder Bar** (`bar_folder_container`) is in the middle.
  - **Files Bar** (`bar_file_container`) is at the bottom.

## 2. Category Label Badges (activity_main.xml)
- Inside the center section of each navigation container, nest the title TextView within a vertical `LinearLayout`.
- Add a small uppercase category header TextView:
  - Text: `"PLAYLIST"`, `"FOLDER"`, or `"FILES"`.
  - Color: Accent Vibrant (`@color/accent_vibrant`) with `80%` opacity (`alpha="0.8"`).
  - Text Size: `11sp` bold.
- Place the main name TextView underneath each header to clearly display the active item.
