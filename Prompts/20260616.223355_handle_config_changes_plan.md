# Handle Configuration Changes Plan

This plan details the changes to add `android:configChanges` to all activities in `AndroidManifest.xml`. This prevents activity recreation on orientation/screen size changes, ensuring that the audio track doesn't change or interrupt when tilting the device.

## User Review Required

> [!NOTE]
> We will add the attribute `android:configChanges="orientation|screenSize|screenLayout|keyboardHidden"` to all activities defined in `AndroidManifest.xml`.
> This prevents Android from destroying and recreating the activities during device rotation, thereby keeping the MediaPlayer instance alive and playing uninterrupted.
